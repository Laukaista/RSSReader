# RSSReader
### Реализовано:
    - Загрузка и отображение новостей Lenta.ru
    - Сохранение новостей в core data
    - Pull to refresh
    - Отметка "просмотрено"
    - Скролл на странице новости
    - Возможность поделиться ссылкой на новость
    - Темная/светлая тема
    - Русская/английская локализация
